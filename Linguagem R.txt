ggplot():
	data = - Nome do banco

	mapping = aes(x =, y =) - Elemento do eixo x e do eixo y

	+ geom_point() - Pontos

	+ geom_smooth() - Linha de tendência

	+ geom_histogram() - Histograma

	+ geom_bar(
		     position = 'fill' - Torna as barras proporcionais
		  ) - Barras 

	+ labs(title, subtitle, x, y, color, shape, etc) - Texto

	+ scale_x_log10() - Escala logarítmica

	+ geom_segment(aes(y = 0, yend = x)) - Linha que vai de 0 até o valor x

	+ scale_x_discrete(labels = c('x' = 'y')) - No eixo x, troca o nome do valor x para y

	+ guides(x = FALSE) - Ocultar legenda x

	+ transition_time(x) - Definir a variação do tempo como a variável temporal x

	+ ease_aes('linear') - Definir uma transição suave

	aes(
		color - Cor

		shape - Formato

		fill - Cor de preenchimento

		alpha - Transparência

		linewidth - Largura da linha

		text = paste() - Escrever informações que serão exibidas em um gráfico interativo
	)

Project R ou Quarto Document (visual):
	/ - Chamar tags e elementos

	$ $ - Escrever operações matemáticas

	{r} - Chunk de código em Linguagem R

	{r x, y=FALSE} - Definir o nome como x, omitir y (também é possível usar #|)

Quarto Document Dashboard (source):
	format: dashboard
		theme: - Alterar o tema
			light: - Definir tema claro
			dark: Definir tema escuro
		
		nav-buttons: - Botões para navegação
			- icon: - Nome do ícone
				href: Link
				title: Título
		
		scrolling: true - Permitir rolagem na tela 

	Hierarquia (do maior para o menor): #  ##  ###  ####
		# x {parâmetros} - Criar página x
		
		## Row {parâmetros} - Criar linha

		## Column {parâmetros} - Criar coluna

		## {.sidebar parâmetros} - Criar sidebar

		## Column | ## Row {.tabset parâmetros} - Criar coluna ou linha com abas

	Parâmetros:
		width=x% - Largura irá ocupar x% da tela

		height=x% - Altura irá ocupar x% da tela

		orientation='rows | columns' - Organização primária da página em linhas ou colunas

		scrolling='false' - Impedindo a rolagem da tela

	Chunk r:
		#| title: - Título

		#| content: - Conteúdo (exemplo: valuebox)

Atalhos:
	alt + hífen (<-)
	
	control + alt + i ({r})

	control + alt + k (knit)

	control + shift + m (|>)

	control + shift + c (comentário)

	control + shift + 1 (2, 3 ou 4) (maximizar janela)

	control + shift + d (dulplicar linha)
