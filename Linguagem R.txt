Linguagem R:  

Raiz:
	library() - Importar bibliotecas
	
	str() - Mudar estrutura de visualização
	
	x$y - Acessar, no banco x, a coluna y

	mean(x, na.rm = T) - Média do valores de x

	z[x, y] - Acessar, no banco z, o elemento em (x, y) 

	print(x, n = y) - Exibir as y primeiras colunas do banco

	c() - Criar lista

	data.frame() - Exibir o banco como tabela

	hist() - Exibir como histograma

	read_csv('x.csv') - Importar banco de dados externo x

	?x - Ajuda 

	head(x) ou glimpse(x) - Pré-visualização do banco x

	filter() - Filtrar linhas

	select() - Selecionar colunas

	mutate() - Criar ou modificar colunas

	x <- x |> mutate() - Modificação definitiva no banco

	drop_na(x) - Desconsidera NA da coluna x

	sd(x) - Desvio padrão dos valores de x

ggplot():
	data = - Nome do banco

	mapping = aes(x =, y =) - Elemento do eixo x e do eixo y

	+ geom_point() - Pontos

	+ geom_smooth() - Linha que se adapta

	+ geom_histogram() - Histograma

	+ geom_bar() - Barras 

	+ labs(title, subtitle, x, y, color, shape, etc) - Texto

	+ scale_x_log10() - Escala logarítmica

	+ geom_segment(aes(y = 0, yend = x)) - Linha que vai de 0 até o valor x

	+ scale_x_discrete(labels = c('x' = 'y')) - No eixo x, troca o nome do valor x para y

	aes(
		color = - Cor

		shape = - Formato

		fill = - Cor de preenchimento

		alpha = - Transparência

		linewidth - Largura da linha
	)

Project R ou Quarto Document (visual):

	{r} - Chunk de código em Linguagem R

	{r x, y=FALSE} - Definir o nome como x, omitir y (também é possível usar #|)
